# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-27

### Added

- Playbook for setting up macOS hosts with Amazon Corretto
- Playbook for installing deviceConnect on macOS hosts
- Playbook for installing gigacap on Ubuntu hosts
- Example inventory file with `mac` and `gem` host groups
- CI workflow for ansible-lint
- Dependabot configuration for GitHub Actions
